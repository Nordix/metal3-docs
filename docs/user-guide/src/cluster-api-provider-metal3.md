# Cluster-api-provider-metal3

[Cluster API Provider Metal³][CAPM3] (CAPM3) provides Kubernetes APIs to integrate CLuster API
worklow with Metal³ project.

## Compatibility with Cluster API

Cluster-api-provider-metal3's versions are compatible with the following versions of Cluster API

| CAPM3 version  | Cluster API version | CAPM3 Release |
| -------------- | ------------------- | ------------- |
| v1alpha2       | v1alpha2            | v0.2.X        |
| v1alpha3       | v1alpha3            | v0.3.X        |
| v1alpha4       | v1alpha4            | v0.4.X        |


[CAPM3]: https://github.com/metal3-io/cluster-api-provider-metal3
